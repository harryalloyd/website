Billboard Top 100 Playlist Creator

Created a web application that scrapes the Billboard Top 100 chart and generates a playlist with the songs on Spotify. It automates this process using the Spotify API to save time and simplify music discovery.

Exercise Tracker

Built a tool that lets users track their exercises by describing them in natural language. The app records workout data into a spreadsheet using the Nutritionix and Sheety APIs, making it easier to manage fitness routines.

ISS Tracker

Developed a script that tracks the location of the International Space Station and sends email alerts when it's visible overhead at night. This project uses the Open Notify API for ISS location data and the Sunrise-Sunset API for night-time checks.

Sudoku Solver

Built a Sudoku solver with multiple levels of difficulty. It includes an automatic solver that completes puzzles for you using a backtracking algorithm to find solutions quickly and accurately.

Battleship

Built a real-time multiplayer Battleship game where players can compete against each other online. The game includes a clean interface, real-time updates, and turn-based mechanics that replicate the classic board game.

Wordle

Created a clone of the popular word-guessing game Wordle. Players guess a five-letter word within six tries, receiving color-coded hints for correct letters and positions. This version offers a clean interface similar to the original game.
